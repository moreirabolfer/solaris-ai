## Sobre o projeto

O Solaris AI Control é um sistema inteligente de monitoramento desenvolvido para uma missão espacial experimental.

O projeto utiliza programação e inteligência artificial introdutória para analisar dados simulados dos módulos da operação, utilizando conceitos de energia renovável, sustentabilidade e controle automatizado.



## Objetivo

Monitorar sistemas energéticos espaciais através da análise de:

- Temperatura dos módulos
- Energia renovável disponível
- Comunicação da missão
- Status operacional



## Funcionamento

O sistema recebe dados simulados de diferentes ciclos da missão.

A inteligência artificial analisa as informações recebidas e classifica o estado da operação.

Possíveis resultados:

- Missão segura
- Risco moderado
- Risco alto



## Inteligência Artificial

A IA foi criada utilizando estruturas de decisão.

Ela identifica alterações nos sensores e gera respostas automáticas para proteger a missão.

Exemplos:

- Ativação de economia de energia
- Uso de painéis solares
- Controle de temperatura
- Ajustes de comunicação



## Ciclo crítico
